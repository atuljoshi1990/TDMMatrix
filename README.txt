Files from 1-45 and stopWords.txt are the input files. 

TDMMatrix.csv, WordFrequency_File_1.csv, ... , WordFrequency_File_45.csv are the output files.

GenerateTDMMatrix.java contains the source code.

To run the program, all the input files and the GenerateTDMMatrix.java file should be in the same directory. 
To run the program execute the following command:

javac *.java
java GenerateTDMMatrix
